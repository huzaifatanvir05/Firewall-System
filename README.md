# Firewall System

A console-based Firewall System developed in C++ using Object-Oriented Programming (OOP) principles. The project simulates core firewall functionalities including traffic filtering, rule management, and access control while demonstrating modular software design and file-based data handling.

## Key Features

- Traffic filtering simulation
- Firewall rule creation and management
- Access control implementation
- Menu-driven user interface
- Persistent data storage using file handling
- Modular and structured system design

## Technologies & Tools

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- File Handling (`fstream`)
- Data Structures
- Vectors
- Strings
- Algorithms
- Modular Programming

## OOP Concepts Applied

- Classes & Objects
- Encapsulation
- Abstraction
- Modular Design

## Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/Firewall-System.git
```

### Compile

```bash
g++ *.cpp -o firewall
```

### Run

```bash
./firewall
```

## Project Objective

The purpose of this project is to simulate fundamental firewall operations while applying practical software engineering concepts and strengthening C++ development skills.
